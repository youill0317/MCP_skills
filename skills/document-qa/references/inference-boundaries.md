# Inference Boundaries

Use this reference when the answer requires interpretation beyond direct quotation.

## Support Types

1. `direct`: the source explicitly states the answer.
2. `partial`: the source supports part of the answer but not all requested detail.
3. `inference`: the answer follows from a narrow, defensible interpretation of the source.
4. `unsupported`: the source does not justify the claim.

## Confidence Ceilings

1. `direct` support from multiple consistent sources may be `High`.
2. A single `direct` source is usually at most `Medium` unless it is highly authoritative and unambiguous.
3. `partial` support should not exceed `Low`.
4. `inference` should not exceed `Medium`, and should usually be `Low` if ambiguity remains.
5. `unsupported` must not be presented as an answer.

## Writing Rules

1. Label inferential reasoning explicitly.
2. Do not mix direct support and inference in a way that hides uncertainty.
3. If only partial support exists, state exactly what remains unconfirmed.
