# Output Contract

Load this reference when the final formatting must be paste-ready.

## Required Shape

```markdown
# {Note Title}

## {Section Title}

### 1. Multiple Choice
Question: ...

A. ...
B. ...
C. ...
D. ...

> [!answer]- Answer and Explanation
> Answer: B
> Explanation: ...

### 2. Fill in the Blank
Question: ...

> [!answer]- Answer and Explanation
> Answer: ...
> Explanation: ...
```

## Formatting Rules

- Keep each answer block directly below its question.
- Put supporting code, tables, formulas, or excerpts before the answer block.
- Keep headings simple and sequential.
- Match the note's dominant language for labels when possible.
- Do not add extra meta commentary outside the practice set unless the user asked for it.
