# Question Construction

Load this reference when question wording must be rigorous and exam-like.

## Multiple-Choice Rules

1. Make one option clearly best from the source evidence.
2. Use 4 options by default unless the source shape strongly suggests otherwise.
3. Build distractors from nearby concepts, not unrelated nonsense.
4. Avoid trivial wording cues such as length imbalance or one overly specific option.
5. If a code or table fragment is required, show only the minimum needed snippet.

## Fill-In-The-Blank Rules

1. Blank the smallest unit that tests meaningful recall.
2. Prefer one blank per sentence unless parallel structure demands two.
3. Preserve enough surrounding context to avoid ambiguity.
4. Use exact source terminology when the note teaches a formal term.
5. If multiple answers could fit, rewrite the sentence before finalizing.

## Explanation Rules

- State the correct answer first.
- Explain why it is correct using the source concept.
- For multiple-choice, briefly note why the most tempting distractor is wrong when useful.
- For fill-in-the-blank, clarify the phrase, rule, or distinction the learner should remember.
