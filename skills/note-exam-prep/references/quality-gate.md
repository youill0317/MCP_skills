# Quality Gate

Load this reference before returning the final practice set.

## Final Checks

1. Every meaningful source item is covered at least once.
2. Both question types are used when the note content supports both.
3. No question depends on unstated outside knowledge.
4. Each multiple-choice item has one best answer.
5. Each fill-in-the-blank item has one unambiguous expected answer.
6. Each answer block includes both `Answer` and `Explanation` or their equivalent in the note's main language.
7. Supporting code, formulas, tables, or excerpts are present when needed to solve the question.
8. Question order follows the source note structure.
9. No duplicate or near-duplicate questions remain.
10. The set reads as a complete exam-prep artifact, not as scattered notes.
