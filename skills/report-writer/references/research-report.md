# Research Report Template

Use this template when converting research outputs into a formal report.

## 1. Executive Summary

- Write 3-5 sentences.
- State the main conclusion first.
- Include the most decision-relevant finding with citation markers.

## 2. Background and Scope

- Define the problem statement and decision context.
- Clarify scope boundaries:
  - included topics
  - excluded topics
  - time window
  - geography or market scope

## 3. Methodology

- Summarize how evidence was gathered and validated.
- Include whether cross-language checks were performed.
- Mention confidence boundaries of the input dataset.

## 4. Key Findings

- Group findings into logical subsections.
- Attach citation markers to each material claim.
- Use a comparison table when contrasting options.

Suggested table:

```markdown
| Topic | Finding | Evidence | Confidence |
|---|---|---|---|
| ... | ... | [1][2] | High/Medium/Low |
```

## 5. Limitations and Risks

- Separate known limitations from open risks.
- List unresolved conflicts between sources.
- Explain potential impact if each risk is ignored.

## 6. Conclusion and Next Steps

- Restate the recommended direction.
- Provide concrete next actions as numbered items.
- Identify additional research questions when evidence is insufficient.

## Writing Constraints

- Keep terminology consistent across sections.
- Use factual language and avoid unsupported claims.
- Use inline citations like `[1][2]` only.
- Do not include external URLs.
