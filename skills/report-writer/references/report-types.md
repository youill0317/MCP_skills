# Report Types

Structure guides for each supported report type.

## Research Report

Full investigation report with comprehensive evidence.

### Structure

1. **Executive Summary** (3-5 sentences)
2. **Background and Scope** (problem, boundaries, time window)
3. **Methodology** (how evidence was gathered)
4. **Key Findings** (grouped by theme, cited)
5. **Limitations and Risks** (gaps, conflicts, open items)
6. **Conclusion and Next Steps** (recommendations, follow-ups)

### Length Target

- 1,500-3,000 words for medium complexity topics.
- Scale up for highly complex investigations.

---

## Executive Brief

Decision-focused summary for leadership.

### Structure

1. **Bottom Line** (1-2 sentences with the main recommendation)
2. **Context** (why this matters, 2-3 sentences)
3. **Key Evidence** (3-5 bullets, each cited)
4. **Risks** (2-3 bullets)
5. **Recommended Action** (specific next steps)

### Length Target

- 300-600 words. One page maximum.

### Tone

- Direct and action-oriented. Lead with conclusions.

---

## Technical Report

Detailed findings for engineering audiences.

### Structure

1. **Summary** (objective and key result)
2. **Technical Context** (architecture, stack, constraints)
3. **Analysis** (detailed findings with technical specifics)
4. **Implementation Notes** (code patterns, config, migration steps)
5. **Trade-offs** (alternatives considered, pros/cons)
6. **Next Steps** (action items with technical detail)

### Length Target

- Variable. Prioritize completeness over brevity.

### Tone

- Precise and technical. Include specifics.

---

## Comparison Report

Side-by-side option analysis for decision-making.

### Structure

1. **Objective** (what is being compared and why)
2. **Evaluation Criteria** (dimensions and their weights)
3. **Comparison Table** (options × criteria matrix)
4. **Analysis** (per-option assessment with evidence)
5. **Recommendation** (preferred option with rationale)
6. **Caveats** (conditions that could change the recommendation)

### Length Target

- 800-1,500 words plus comparison table.

### Tone

- Balanced and objective. Present all sides fairly.
