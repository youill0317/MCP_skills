# Copy Hierarchy

Use this reference when the brief needs stronger message structure.

## Order of Importance

1. Headline: the first thing the viewer must notice.
2. Subheadline: the clarifier that adds context or promise.
3. Support copy: short evidence, details, or value proposition.
4. CTA: the next action the viewer should take.

## Headline Rules

- Keep it short and scannable.
- Prefer one idea, not a compound sentence.
- Make the benefit, event, or offer legible immediately.

## Subheadline Rules

- Add missing context without repeating the headline.
- Clarify who, what, or why in one short line.

## Support Copy Rules

- Use compact facts, not paragraphs.
- Prioritize dates, times, offers, proof points, and practical details.

## CTA Rules

- Make the action explicit.
- Match the channel: register, visit, swipe, shop, learn more, RSVP.
- Do not bury the CTA under secondary information.
