# Quality Gate

Run this before finalizing the design brief.

## Message Checks

1. Is there exactly one primary message?
2. Is the headline stronger than the support copy?
3. Is the CTA visible and action-oriented?
4. Are must-include details present?

## Layout Checks

1. Does the layout match the asset type?
2. Is the information density realistic for the format?
3. Is there a clear reading order?

## Risk Checks

1. Remove duplicated copy.
2. Flag vague claims that need evidence.
3. Flag overcrowded layouts.
4. Flag missing dates, links, prices, or contact details when required.
