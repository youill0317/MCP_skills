# Quality Gate

Run this checklist before returning a final deck spec.

## A. Narrative Checks

1. Deck has one explicit takeaway sentence.
2. Section order follows the selected deck type.
3. Each slide contributes to the final decision or learning objective.
4. Closing slide includes explicit action, decision, or retention point.

## B. Slide Clarity Checks

1. Every slide has one key message sentence.
2. Slide title and key message are aligned.
3. Content blocks are concise and non-redundant.
4. Speaker notes clarify assumptions and transitions.

## C. Visual System Checks

1. Typography hierarchy is consistent across slides.
2. Color roles are consistent and readable.
3. Layout alignment and spacing are coherent.
4. Visual emphasis supports the intended message.

## D. Data Integrity Checks

1. Claims with numbers have sufficient context and evidence.
2. Chart types match the analytical question.
3. Axes, scales, and units do not mislead.
4. Unverified claims are tagged `[EVIDENCE NEEDED]`.

## E. Delivery Readiness Checks

1. Slide count fits available time.
2. Opening and closing are audience-appropriate.
3. Major transitions are explicit.
4. Audience decision or takeaway is unambiguous.

## Quality Gate Output

Return:

1. `Passed checks`
2. `Failed checks`
3. `Required fixes`
4. `Assumptions used`
