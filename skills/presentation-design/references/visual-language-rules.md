# Visual Language Rules

Define consistent visual grammar before detailing individual slides.

## Typography Hierarchy

1. Use three levels only:
   - Level 1: slide title
   - Level 2: section labels or key numbers
   - Level 3: supporting body text
2. Keep consistent hierarchy across all slides.
3. Avoid decorative font changes by slide.

## Color Roles

1. Assign semantic roles, not random colors:
   - primary: core message emphasis
   - neutral: structure and baseline context
   - accent: one callout or highlight
2. Keep contrast high for readability.
3. Avoid using color alone to encode critical meaning.

## Layout and Spacing

1. Use a stable alignment grid.
2. Maintain whitespace around key content blocks.
3. Keep visual density balanced; remove non-essential decoration.

## Imagery and Icons

1. Use visuals only when they clarify a message.
2. Prefer simple, topic-relevant visuals over generic stock style.
3. Keep icon style consistent within one deck.
4. Do not use visuals that imply unsupported conclusions.

## Accessibility and Clarity

1. Ensure text remains legible at typical presentation distance.
2. Keep wording concise and concrete.
3. Avoid jargon unless audience expertise is confirmed.
4. Add spoken clarification in notes for nuanced visuals.

## Visual Anti-Patterns

1. Title that repeats content without adding meaning.
2. Multiple competing focal points on one slide.
3. Decorative backgrounds that reduce contrast.
4. Over-animated flow that distracts from argument.
