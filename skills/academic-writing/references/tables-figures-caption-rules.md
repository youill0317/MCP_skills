# Tables, Figures, and Captions (APA 7)

Apply these rules whenever tables or figures are present.

## Table Rules

1. Number tables in order of appearance: `Table 1`, `Table 2`, ...
2. Put table number on one line and table title on the next line.
3. Use concise, descriptive titles.
4. Mention each table in text before it appears.
5. Include notes under table when needed:
- General note for explanation.
- Specific note for marked cells.
- Probability note for p-value symbols.

## Figure Rules

1. Number figures in order of appearance: `Figure 1`, `Figure 2`, ...
2. Provide clear figure titles and captions.
3. Ensure axis labels, legends, and units are unambiguous.
4. Mention each figure in text before it appears.
5. Add source attribution in note when figure is adapted.

## Caption Quality Checks

1. Ensure numbering is gap-free and unique.
2. Ensure every referenced table/figure exists.
3. Ensure every existing table/figure is referenced in text.
4. Keep terminology consistent with the manuscript body.
