# APA Paper Profiles and Required Elements

Use this file to decide which APA paper profile applies and which front-matter elements are required.

## Supported APA Paper Profiles

1. `student-paper`
2. `professional-manuscript`
3. Generic APA 7 when the user does not specify a profile

## Student Paper

1. Require a title page with title, author name, institutional affiliation, course information, instructor, and due date when provided.
2. Page number is typically required.
3. Running head is usually not required.
4. Abstract and keywords may be required depending on the assignment; if unclear, flag as an assumption.

## Professional Manuscript

1. Require a title page with title, author names, affiliations, author note when available, and correspondence details if provided.
2. Page number is required.
3. Running head may be needed depending on the specific APA interpretation used by the user; if unavailable, mark as an assumption rather than inventing it.
4. Abstract is normally expected.
5. Keywords are included when the manuscript design calls for them.

## Generic APA 7 Default

1. If the user does not specify a profile, use generic APA 7 manuscript assumptions.
2. Include the front matter that can be justified from the provided materials.
3. Record any uncertain profile-specific requirements in the assumptions block.

## Required-Elements Check

1. Identify which APA profile is active before drafting or reviewing.
2. List required elements explicitly before the manuscript body.
3. Do not fabricate title-page details that the user did not provide.
4. If a required element is missing, report it in `Missing Materials or Fix Requests`.
