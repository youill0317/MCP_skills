# Quotations, Footnotes, and Appendices

Apply these rules whenever these elements appear.

## Direct Quotations

1. Prefer paraphrase unless the exact wording matters.
2. Include page or paragraph locators for all direct quotations.
3. Distinguish short quotations from block quotations using APA conventions.
4. Do not quote text that cannot be traced to a provided source.

## Footnotes

1. Use footnotes sparingly.
2. Keep footnotes supplemental rather than essential to the core argument.
3. Number footnotes consistently when used.
4. If the user did not provide footnote content, do not invent it.

## Appendices

1. Label appendices clearly and sequentially.
2. Reference each appendix in the main text.
3. Keep appendix titles descriptive and aligned with the manuscript terminology.
4. Report missing appendix content explicitly rather than implying it exists.

## Review Checks

1. Every direct quotation has a locator.
2. Footnotes are necessary, consistent, and non-duplicative.
3. Every appendix is labeled, referenced, and scoped clearly.
