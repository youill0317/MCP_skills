# Input Sufficiency and Gap Handling

Use this file to decide whether drafting or review can proceed safely.

## Materials to Check

1. Working title or topic.
2. Thesis, research question, or paper objective.
3. Source list or source excerpts for cited claims.
4. User-provided data, results, or analysis outputs when findings are reported.
5. Existing draft text when the task is revision or APA review.
6. Tables, figures, appendices, title-page details, and author metadata when relevant.

## Sufficiency Rules

1. Proceed with drafting when the core topic, paper objective, and enough supporting materials are present.
2. Proceed with APA review when the manuscript text is present, even if some metadata is missing.
3. Do not invent missing materials in order to force a complete draft.
4. Use explicit assumptions only for non-substantive formatting ambiguities.

## Gap Tags

1. Use `[CITATION NEEDED]` for unsupported external or prior-work claims.
2. Use `[MISSING METADATA: field]` for incomplete references or front-matter details.
3. Use `[VERIFY SOURCE]` when source identity or reliability is uncertain.

## Missing-Materials Reporting

1. List each missing item separately.
2. Explain whether the gap blocks drafting, weakens the draft, or prevents final APA verification.
3. Request the smallest concrete input that would resolve the gap.
