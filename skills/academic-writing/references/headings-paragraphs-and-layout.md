# Headings, Paragraphs, and Layout

Apply these rules to the visible manuscript formatting guidance.

## Headings

1. Use APA-style heading levels consistently.
2. Do not skip heading levels.
3. Keep sibling headings parallel in grammar and scope.
4. Use headings to describe content, not to decorate.

## Paragraphs

1. Keep one coherent claim cluster per paragraph.
2. Use consistent first-line indentation unless the element type calls for a different format.
3. Avoid abrupt one-sentence paragraphs unless the rhetorical purpose is clear.
4. Define abbreviations at first use and keep terminology stable.

## Layout Guidance

1. Preserve APA-consistent spacing, alignment, and page-flow guidance in the written instructions.
2. Keep block quotations visually distinct from regular paragraphs.
3. Keep tables, figures, notes, and appendices separated clearly from body text.
4. When exact editor styling cannot be verified from plain text, state the expected APA layout requirement in the compliance report.

## Review Checks

1. Heading hierarchy is stable.
2. Paragraph formatting instructions are internally consistent.
3. Quotations and special blocks are clearly differentiated from normal prose.
