# Type Guide: Methodological Paper

Use this structure when proposing, validating, or comparing methods.

## Target Section Order

1. Title
2. Abstract
3. Introduction
4. Method Rationale
5. Method Design or Instrument Development
6. Validation Strategy
7. Results
8. Practical Guidance and Limitations
9. References

## Writing Priorities

1. Explain why current methods are insufficient.
2. Document reproducible method steps.
3. Specify reliability and validity evidence.
4. State operational constraints and expected failure conditions.
5. Provide implementation guidance without overclaiming.

## Common Failure Modes

1. New method introduced without baseline comparison.
2. Validation evidence missing or weakly specified.
3. Practical limits omitted in conclusion.
