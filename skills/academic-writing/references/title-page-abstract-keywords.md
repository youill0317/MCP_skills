# Title Page, Abstract, and Keywords

Apply these rules when the manuscript includes front matter.

## Title Page

1. Use a concise, descriptive title that matches the manuscript body.
2. Place title-page elements in a stable, APA-consistent order.
3. Keep author and affiliation lines separate and unambiguous.
4. Do not invent course names, departments, correspondence addresses, or author-note details.

## Abstract

1. Write the abstract as a stand-alone summary of the manuscript.
2. Keep the abstract aligned with the manuscript body and conclusion.
3. Avoid introducing claims that are not supported in the main text.
4. If the assignment does not require an abstract, note that decision explicitly.

## Keywords

1. Include keywords only when required or clearly expected for the active profile.
2. Keep keywords brief and concept-specific.
3. Ensure keywords match the paper's terminology and scope.

## Review Checks

1. Title page is complete for the active APA profile.
2. Abstract reflects the actual content of the manuscript.
3. Keywords are present, absent, or flagged with a clear rationale.
