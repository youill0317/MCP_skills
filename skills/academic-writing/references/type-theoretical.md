# Type Guide: Theoretical Paper

Use this structure when the paper advances or refines conceptual models.

## Target Section Order

1. Title
2. Abstract
3. Introduction
4. Conceptual Background
5. Theory Development or Integration
6. Propositions or Model Implications
7. Limitations and Future Work
8. References

## Writing Priorities

1. Define core constructs before argument expansion.
2. State boundary conditions for each proposition.
3. Distinguish original contribution from prior theory.
4. Tie conceptual claims to supporting literature.
5. Avoid unsupported universal statements.

## Common Failure Modes

1. Construct definitions drift across sections.
2. Claims presented as theory without grounding sources.
3. Missing explanation of novelty versus existing models.
