# Type Guide: Mixed-Methods Paper

Use this structure when combining quantitative and qualitative strands.

## Target Section Order

1. Title
2. Abstract
3. Introduction
4. Mixed-Methods Design Rationale
5. Quantitative Strand Method and Results
6. Qualitative Strand Method and Results
7. Integration and Meta-Inference
8. Limitations and Implications
9. References

## Writing Priorities

1. Name the mixed-methods design explicitly (convergent, explanatory sequential, exploratory sequential, etc.).
2. Explain timing, priority, and integration points.
3. Keep strand-specific methods independently auditable.
4. Show how integrated inference changes interpretation.
5. Resolve or explicitly report cross-strand contradictions.

## Common Failure Modes

1. Two strands presented independently without integration logic.
2. Integration occurs only in conclusion, not in analysis.
3. Contradictory strand results hidden instead of discussed.
