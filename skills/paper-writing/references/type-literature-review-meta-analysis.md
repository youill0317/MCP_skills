# Type Guide: Literature Review / Meta-Analysis

Use this structure for synthesis-first papers.

## Target Section Order

1. Title
2. Abstract
3. Introduction
4. Review Method (search/selection/coding)
5. Synthesis of Findings
6. Implications and Gaps
7. References

For meta-analysis, add:

8. Effect Size Model and Heterogeneity
9. Publication Bias Checks

## Writing Priorities

1. Define inclusion and exclusion criteria explicitly.
2. Keep search strategy reproducible.
3. Separate descriptive synthesis from evaluative judgment.
4. Report contradictions and unresolved debates directly.
5. Avoid claiming completeness unless verified.

## Common Failure Modes

1. Opaque source selection logic.
2. Thematic synthesis without methodological transparency.
3. Claims of consensus where evidence is mixed.
