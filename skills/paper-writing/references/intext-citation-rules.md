# In-Text Citation Rules (APA 7)

Apply author-date citations consistently.

## Basic Patterns

1. One author: `(Smith, 2024)` or `Smith (2024)`.
2. Two authors: `(Smith & Lee, 2024)` every time.
3. Three or more authors: `(Smith et al., 2024)`.
4. Group author: `(World Health Organization [WHO], 2023)` first mention, then `(WHO, 2023)`.

## Multiple Sources

1. Put multiple citations in one parenthesis when supporting the same claim.
2. Sort entries alphabetically by first author/group name.
3. Separate sources with semicolons.
4. Keep year suffixes when needed: `(Kim, 2020a, 2020b)`.

## Direct Quotes

1. Include locator information for direct quotes: page or paragraph.
2. Example: `(Smith, 2024, p. 18)` or `(Smith, 2024, para. 4)`.
3. Use quotes sparingly and prefer paraphrase when possible.

## Secondary Citation

1. Prefer reading and citing the primary source.
2. If unavoidable, use `as cited in` pattern and include only the source actually read in references.

## Citation Integrity Rules

1. Attach citation at sentence level for factual claims.
2. Do not stack citations that do not support the exact claim.
3. If support is missing, mark `[CITATION NEEDED]`.
4. If source identity is uncertain, mark `[VERIFY SOURCE]`.
