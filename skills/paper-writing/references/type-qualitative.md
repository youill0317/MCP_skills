# Type Guide: Qualitative Paper

Use this structure for interview, ethnography, grounded theory, phenomenology, or related designs.

## Target Section Order

1. Title
2. Abstract
3. Introduction
4. Positionality and Context
5. Methods (sampling, data collection, analysis)
6. Findings with Evidence Excerpts
7. Trustworthiness and Limitations
8. Discussion
9. References

## Writing Priorities

1. State epistemological stance clearly.
2. Document sampling and recruitment logic.
3. Describe coding and theme-generation process transparently.
4. Connect findings to concrete evidence excerpts.
5. Address credibility, transferability, and reflexivity.

## Common Failure Modes

1. Findings presented without traceable evidence excerpts.
2. Researcher role and bias handling omitted.
3. Method narrative too thin for auditability.
