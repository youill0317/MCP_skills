# Document Summary Workflow

1. Read target documents or references.
2. Identify the main claims, data points, and open questions.
3. Produce a layered summary:
   - 1-line takeaway
   - short paragraph summary
   - key points
4. Add unknowns and follow-up checks when data is incomplete.
