# Summary Style

Load this reference when the summary needs stricter wording, density, or formatting control.

## Content Rules

- Prioritize facts over opinions.
- Keep each bullet atomic: one claim per bullet.
- Use consistent terms for repeated entities throughout the summary.
- If data is missing, state it explicitly rather than omitting silently.
- Distinguish between what the document states and what it implies.

## Language and Tone

- Mirror the user's language unless otherwise requested.
- Use neutral, informative tone.
- Avoid hedging language unless reflecting genuine uncertainty from the source.
- Prefer active voice over passive voice.

## Formatting

- Use Markdown for structure: headers, bullets, bold for key terms.
- Keep bullet points to one line where possible.
- Use numbered lists for sequential or prioritized items.
- Use tables for comparative data when three or more items are compared.

## Length Targets

- 1-line takeaway: max 30 words.
- Summary paragraph: 3-5 sentences, max 100 words.
- Key points: 5-10 bullets for Standard, up to 20 for Detailed.
- Each bullet: max 25 words.
